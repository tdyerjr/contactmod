-----The following information is about the installation process. This was created/tested on Moodle 2.5.3+

* Unzip the archive and read this file

* Place contactmod folder in the /mod folder

* Navigate to Notifications, under Site Administration

* Install the Faculty Contact Module

* Under the custom profile field you'll see that it created a Faculty Information field for the office hours.

* TaDa!! Now you can add this awesome Resource Module to your courses.

-----How to edit the information that displays

* Everything that displays will pull from the profile (Name, Email, Phone Number, Faculty Information/office hours)
